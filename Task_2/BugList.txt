1. (File) Для файлов не имеющих расширения, оно сохраняется не верно.
2. (File) Отсутствует проверка на задание недопустимого имени файла.
3. (FileStorage) Несопоставимая инициализация поля availibleSize конструктором по умолчанию и пользовательским конструктором
4. (FileStorage) Инициализация поля availibleSize несопоставимым значением с переданным в пользовательский конструктор
5. (FileStorage) Ни где не используется поле maxSize, и отсюда допустимо передать файл больший по размеру нежели maxSize.
6. (FileStorage) Из за некорректного задания поля availibleSize в пользовательском конструкторе, допустимо записать в хранилище файлов, коллекцию файлов, превышающую по суммарному размеру, максимально дозволенный.
